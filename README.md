# Portfolio — Samet Kurt

Persönliches Portfolio: Prozessautomatisierung, Datenpipelines und operative Werkzeuge, gebaut mit Python und KI-gestützter Entwicklung.

**Live:** https://lygonian.github.io

## Aufbau

Statische Website ohne Build-Schritt oder Framework — reines HTML und ein gemeinsames Stylesheet, gehostet über GitHub Pages.

```
index.html            Startseite
projekte.html         Projektübersicht
ueber-mich.html       Über mich / Arbeitsweise
ki-workflow.html      KI-Workflow / How-to
forschung.html        Akademischer Hintergrund
kontakt.html          Kontakt
projekte/             Detailseiten der Hauptprojekte
assets/css/style.css  Gemeinsames Stylesheet
pdf/                  Zertifikat und Abschlussarbeiten
```

## Pflege

Design und Struktur liegen zentral in `assets/css/style.css`. Neue Projektseiten entstehen durch Kopieren einer bestehenden Seite aus `projekte/`. Der redaktionelle Prozess (inklusive Anonymisierung und Datenschutz-Prüfung) ist außerhalb dieses Repos dokumentiert.
