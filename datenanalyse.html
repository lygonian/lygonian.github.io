Kundensegmentierung mit RFM und K-Means

Welche Kunden kaufen oft, aber wenig? Wer kauft viel, kommt aber nie wieder? Ein E-Commerce-Unternehmen mit 50.000+ Transaktionen hatte die Daten — aber keine Antworten. Mit RFM-Analyse und unüberwachtem Clustering entstanden fünf klare Kundenprofile. Das Ergebnis: zielgerichtete Kampagnen statt Gießkannenprinzip.

# RFM-Score berechnen und normalisieren
from sklearn.preprocessing import MinMaxScaler
from sklearn.cluster import KMeans

def compute_rfm(df, reference_date):
    rfm = df.groupby('customer_id').agg(
        recency   = ('order_date', lambda x: (reference_date - x.max()).days),
        frequency = ('order_id', 'nunique'),
        monetary  = ('revenue', 'sum')
    ).reset_index()

    scaler = MinMaxScaler()
    rfm[['R', 'F', 'M']] = scaler.fit_transform(rfm[[
        'recency', 'frequency', 'monetary'
    ]])
    rfm['R'] = 1 - rfm['R']  # Inversion: niedrige Recency = besser
    return rfm

def find_optimal_k(data, k_range=range(2, 11)):
    inertias, silhouettes = [], []
    for k in k_range:
        km = KMeans(n_clusters=k, random_state=42, n_init=10)
        labels = km.fit_predict(data)
        inertias.append(km.inertia_)
        silhouettes.append(silhouette_score(data, labels))
    return inertias, silhouettes

E-Commerce Kaufverhalten & Warenkorb-Analyse

Was kaufen Menschen zusammen — und warum? Hinter jedem Einkaufskorb steckt ein Muster. Mit Market Basket Analysis und Assoziationsregeln wurden versteckte Produktkombinationen sichtbar gemacht. Das direkte Ergebnis: Cross-Sell-Empfehlungen mit messbarem Lift auf den Durchschnittswarenkorb.

Apriori mlxtend Association Rules pandas seaborn
Vorgehensweise
1
Transaktionsmatrix aufbauen
Pivot-Transformation: Zeilen = Transaktionen, Spalten = Produkte, Werte = 0/1 (One-Hot-Encoding).
2
Frequent Itemsets mit Apriori
Mindest-Support 2 % — filtert seltene Kombinationen, behält statistisch relevante Muster.
3
Regelgenerierung & Filterung
Confidence ≥ 0.5, Lift ≥ 1.5 — nur Regeln mit echtem Mehrwert gegenüber dem Zufall.
4
Business-Übersetzung
Top-Regeln nach Lift sortiert und in konkrete Empfehlungen für Produktplatzierung und E-Mail-Kampagnen übersetzt.
from mlxtend.frequent_patterns import apriori, association_rules

def build_basket_matrix(df):
    # Transaktionsmatrix erstellen
    basket = (df.groupby(['invoice', 'product'])['quantity']
                .sum()
                .unstack(fill_value=0))
    return basket.applymap(lambda x: 1 if x > 0 else 0)

def extract_rules(basket, min_support=0.02, min_confidence=0.5):
    itemsets = apriori(basket, min_support=min_support, use_colnames=True)
    rules = association_rules(itemsets, metric="confidence",
                                min_threshold=min_confidence)
    return (rules
            .query("lift >= 1.5")
            .sort_values("lift", ascending=False)
            .reset_index(drop=True))

ROSSMANN Umsatzprognose — Kaggle Competition

1.115 Filialen. 2,5 Jahre Verkaufsdaten. Die Frage: Wie viel wird jede Filiale in den nächsten sechs Wochen verkaufen? Das Rossmann-Problem ist ein Klassiker — aber erst mit den richtigen Features (Promotions, Feiertage, Wettbewerb) und XGBoost wird aus Rauschen ein brauchbares Signal.

import xgboost as xgb
import numpy as np

def engineer_time_features(df):
    df['week']           = df['date'].dt.isocalendar().week.astype(int)
    df['month']          = df['date'].dt.month
    df['quarter']        = df['date'].dt.quarter
    df['days_to_holiday'] = df['state_holiday_next'] - df['date']
    df['promo_open']     = df['promo'] * df['open']
    return df

def rmspe_xgb(yhat, dtrain):
    y = dtrain.get_label()
    mask = y != 0
    err = np.sqrt(np.mean(((y[mask] - yhat[mask]) / y[mask]) ** 2))
    return 'rmspe', float(err)

params = {'max_depth': 8, 'eta': 0.05, 'subsample': 0.8,
          'colsample_bytree': 0.7, 'objective': 'reg:squarederror'}
model = xgb.train(params, dtrain, num_boost_round=500,
                   feval=rmspe_xgb, evals=[(dvalid, 'val')],
                   early_stopping_rounds=50, verbose_eval=50)
